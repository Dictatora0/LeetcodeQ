# Leetcode 解题集合

## 文件结构

```
Leetcode/
├── solutions/           # 解题代码
│   └── cpp/            # C++ 解题文件
├── build/              # 编译输出
│   ├── bin/           # 可执行文件
│   └── debug/         # 调试符号文件(.dSYM)
├── scripts/            # 工具脚本
├── notes/              # 解题笔记和思路
└── README.md          # 项目说明
```

## 使用方法

### 编译单个题目

```bash
# 编译并运行（Release，速度优先）
./scripts/run_cpp.sh solutions/cpp/543_二叉树的直径.cpp

# 仅编译不运行
./scripts/run_cpp.sh solutions/cpp/543_二叉树的直径.cpp --build-only

# 调试模式（编译更快、便于调试）
MODE=debug ./scripts/run_cpp.sh solutions/cpp/543_二叉树的直径.cpp
```

### 批量编译

```bash
# 使用脚本批量编译
./scripts/compile_all.sh
```

### 运行测试

```bash
# 运行特定题目
./build/bin/239
```
