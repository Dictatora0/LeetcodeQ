## 📖 每个模板包含的内容

每个模板文件都包含完整的 10 个部分：

1. **算法概述** - 核心思想、通俗解释、为什么有效
2. **识别题型** - 如何快速识别、关键词、题目特征
3. **核心模板** - 可直接复制的代码框架，详细注释
4. **常见变体** - 5-7 种变体及其实现
5. **面试技巧** - 如何向面试官解释、常见追问
6. **易错点** - 常见错误、调试技巧、边界条件
7. **复杂度分析** - 时间空间复杂度、优化技巧
8. **真题实战** - 10-15 道 LeetCode 真题，按难度分类
9. **完整示例** - 2-3 个完整的题目解答，可直接运行
10. **速查表** - 一页纸的核心要点，面试前快速复习

## 🎯 使用建议

### 学习顺序（按难度）
1. 简单：双指针 → 前缀和 → 二分查找
2. 中等：滑动窗口 → DFS → BFS → 单调栈
3. 进阶：回溯 → 动态规划 → 并查集

### 面试准备
1. **第一遍**：理解算法原理，手写核心模板
2. **第二遍**：刷对应的高频题目，形成肌肉记忆
3. **第三遍**：面试前看速查表，快速复习

### 编译运行
每个模板都可以独立编译运行：

```bash
# 编译
g++ -std=c++11 -o sliding_window sliding_window_template.cpp

# 运行
./sliding_window
```
