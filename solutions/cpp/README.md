# 原始代码目录说明

## 目录定位

本目录 (`solutions/cpp/`) 包含**历史版本的原始代码**，主要用于：

1. **版本对比** - 保留最初的解题思路，便于对比学习
2. **快速验证** - 简单的代码实现，可快速编译测试
3. **历史备份** - 记录解题历程

## 与分类目录的关系

- **分类目录** (`solutions/array/`, `solutions/binary_tree/` 等)：
  - 包含完整的文档化代码
  - 详细的中文注释和多种解法
  - 复杂度分析和测试用例
  - **推荐用于学习和面试准备**

- **本目录** (`solutions/cpp/`)：
  - 简洁的原始实现
  - 最初的解题代码
  - 可能包含调试代码和 TODO
  - **仅供参考和对比**

## 使用建议

### 学习新题目
1. 先查看分类目录下的完整文档版本
2. 理解算法思路和多种解法
3. 如需对比，可参考本目录的原始版本

### 快速验证
1. 本目录的代码更简洁，适合快速编译测试
2. 使用 `../scripts/run_cpp.sh` 编译运行

## 文件状态

- ✅ **已迁移** - 对应的分类目录已有完整文档版本
- 📝 **待迁移** - 尚未添加详细文档，仅有原始代码
- 🔄 **同步中** - 正在更新文档

## 注意事项

- 本目录文件可能包含：
  - 未使用的 `#include`
  - 调试用的 `main()` 函数
  - `TODO` 注释
  - 简单的测试代码

- 这些是正常的，因为本目录保留了最初的开发状态

## 迁移计划

逐步将本目录的题目迁移到对应分类目录，添加完整文档后，本目录文件将标记为"已迁移"状态。
